SSH Connection Info

Use the following url to connect to the site with ssh:

ssh.atomicsites.net

Clicking on the copy button next to each user will place the full terminal command to your system clipboard. Example:

ssh -v user123@ssh.atomicsites.net

The following key types and lengths are accepted when creating or updating an SSH user:

  • ssh-rsa – 2048 to 16384 bits
  • ecdsa-sha2-nistp256 – 256 bits
  • ssh-ed25519 – 256 bits
